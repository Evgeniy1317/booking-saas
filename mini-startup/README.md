# Booking SaaS - Vertical SaaS Platform

Онлайн-платформа для управления бизнесом и записями клиентов для салонов, мастерских и сервисов.

## 🚀 Технологии

- **React 18** + **TypeScript**
- **Vite** - быстрая сборка
- **Tailwind CSS** - стилизация
- **React Router** - маршрутизация

## 📦 Установка

```bash
# Установить зависимости
npm install

# Запустить dev сервер
npm run dev

# Собрать для продакшена
npm run build
```

## 🏗️ Структура проекта

```
src/
├── components/
│   └── landing/      # Компоненты лендинга
├── pages/
│   └── Landing.tsx   # Главная страница
├── App.tsx           # Главный компонент
└── main.tsx          # Точка входа
```
