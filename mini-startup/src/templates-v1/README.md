# Шаблоны конструктора — базовая версия (зафиксирована)

Базовые шаблоны **зафиксированы как готовые**. Структура страницы и блоков для обычных шаблонов не меняется. Далее идёт разработка премиум-шаблонов (2 отдельных сайта с другой структурой).

## Реестр шаблонов

- **Обычные шаблоны (v1, зафиксированы):** `src/lib/template-registry.ts` — `ORDINARY_THEME_IDS`, `BASE_BLOCK_IDS`
- **Премиум:** текущие `premium-hair` / `premium-barber` пока используют ту же структуру; 2 новых премиум-сайта — своя структура (добавлять отдельно)

## Где лежит реализация

- **Конструктор (редактирование):** `src/pages/ConstructorPage.tsx`, `src/pages/dashboard/Settings.tsx`
- **Публичный сайт (отображение):** `src/pages/PublicPage.tsx`
- **Маршрут публичного сайта:** `/b/:slug` (App.tsx)

## Структура блоков (базовая — не менять для обычных шаблонов)

Порядок блоков в конструкторе и на сайте:

1. **Шапка сайта** (`header`) — темы: Парикмахерская, Барбершоп, Косметология, Покраска волос, Маникюр
2. **Фотографии салона** (`gallery`) — 5 слотов (publicWorks), сетка 1+4
3. **Запись клиентов** (`booking`)
4. **Галерея работ** (`works`) — карусель, до 10 фото (publicGallery)
5. **Карта и адрес** (`map`)
6. **Контактная информация** (`footer`)

## Данные (localStorage)

- Публичные ключи: `publicName`, `publicTagline`, `publicWorks1`…`publicWorks5`, `publicGallery1`…`publicGallery10`, `publicSectionVisibility`, `publicHeaderTheme`, и др.
- Черновики: `draft_*_${slug}_${theme}` для несохранённых изменений (по салону и теме).

## Премиум-шаблоны (далее)

- Текущие премиум (premium-hair, premium-barber) — та же базовая структура.
- Планируются **2 новых премиум-сайта** с другой структурой (свои страницы/роуты, не BASE_BLOCK_IDS).
