# Шаблоны конструктора v1 (сохранённая версия)

Эти шаблоны соответствуют текущему конструктору до перехода на конструктор v2.

## Где лежит реализация

- **Конструктор (редактирование):** `src/pages/dashboard/Settings.tsx`
- **Публичный сайт (отображение):** `src/pages/PublicPage.tsx`
- **Маршрут публичного сайта:** `/b/:slug` (App.tsx)

## Структура блоков (скелет v1)

Порядок блоков в конструкторе и на сайте:

1. **Шапка сайта** (`builder-header`) — темы: Кастомный, Парикмахерская, Барбершоп, Косметология, Покраска волос, Маникюр
2. **Задний фон тела** (`builder-body-bg`)
3. **Фотографии салона** (`builder-gallery`) — 5 слотов (publicWorks), сетка 1+4
4. **Запись клиентов** (`builder-booking`)
5. **Галерея работ** (`builder-works`) — карусель, до 10 фото (publicGallery)
6. **Карта и адрес** (`builder-maps`)
7. **Контактная информация** (`builder-footer`)

## Данные (localStorage)

- Публичные ключи: `publicName`, `publicTagline`, `publicWorks1`…`publicWorks5`, `publicGallery1`…`publicGallery10`, `publicSectionVisibility`, `publicHeaderTheme`, и др.
- Черновики: `draft_*` для несохранённых изменений.

## Восстановление

Текущий код не удалён. Для отката к этой версии используйте историю git до коммитов конструктора v2.
